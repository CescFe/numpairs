# PRD - Tetonor Android App (v0)

## Goal
Build a simple Android app that allows users to view and play a Tetonor puzzle.

## Target User
- Casual puzzle players
- People interested in mental math games

## MVP Scope
- Single screen
- Display:
    - 8 grid with results
    - 8-number strip
- Local editing feedback for fully-known incorrect tiles
- Automatic prevention of incorrect tile assignments
- Whole-puzzle validation and completion flow
- No scoring
- No persistence
- No animations

## Out of Scope
- Difficulty levels
- Timer
- User accounts

## Success Criteria
- App renders correctly
- UI is understandable
- Ready for iteration
