# PRD - NumPairs (v0 Historical Snapshot)

> Historical note: this document captures the original playable prototype scope. It remains as a historical snapshot and was later superseded by [prd-v1.md](./prd-v1.md) and [prd-v2.md](./prd-v2.md).

## Goal
Build a simple Android app that allows users to view and play a NumPairs puzzle.

## Target User
- Casual puzzle players
- People interested in mental math games

## MVP Scope
- Single screen
- Display:
    - 8 grid with results
    - 8-number strip
- Local editing feedback for fully-known incorrect tiles
- Automatic prevention of incorrect tile assignments
- Whole-puzzle validation and completion flow
- No scoring
- No persistence
- No animations

## Out of Scope
- Difficulty levels
- Timer
- User accounts

## Success Criteria
- App renders correctly
- UI is understandable
- Ready for iteration
