# PRD - Tetonor Android App (v0)

## Goal
Build a simple Android app that allows users to view and play a Tetonor puzzle.

## Target User
- Casual puzzle players
- People interested in mental math games

## MVP Scope
- Single screen
- Display:
    - 4x4 grid with results
    - 16-number strip
- No game validation
- No scoring
- No persistence
- No animations

## Out of Scope
- Game logic validation
- Difficulty levels
- Timer
- Backend
- User accounts

## Success Criteria
- App renders correctly
- UI is understandable
- Ready for iteration
